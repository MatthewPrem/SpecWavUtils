# Spec Utils
A collection of helpful functions I have written to help me analyze 1D and 2D spectra and fit lines.
Contains:
- Functions to load and save line lists
- Functions to crop and flip, and display images
- Interactive code to create a linear conversion between pixel space and wavelength space
- A continuum subtraction function
- Interactive code to fit Lorentzian or Gaussian lines to a 1D spectrum and calculate R values.
